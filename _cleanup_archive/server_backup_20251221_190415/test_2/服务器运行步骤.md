# 服务器运行 Flask 应用 - 快速步骤

## 🚀 方法一：systemd 服务（最推荐）⭐

### 首次部署步骤：

```bash
# 1. SSH 连接到服务器
ssh -i "C:\Users\Abraham\Downloads\KeyPair-6418.pem" -p 2950 root@60.10.230.156

# 2. 进入项目目录
cd /var/www/html

# 3. 安装 systemd 服务（如果已有 install_service.sh 脚本）
chmod +x install_service.sh
sudo ./install_service.sh

# 或者手动安装：
sudo cp flask-app.service /etc/systemd/system/
sudo systemctl daemon-reload
sudo systemctl enable flask-app.service
sudo systemctl start flask-app.service

# 4. 检查服务状态
sudo systemctl status flask-app.service
```

### 日常管理命令：

```bash
# 启动服务
sudo systemctl start flask-app.service

# 停止服务
sudo systemctl stop flask-app.service

# 重启服务（上传新文件后需要重启）
sudo systemctl restart flask-app.service

# 查看服务状态
sudo systemctl status flask-app.service

# 查看实时日志
sudo journalctl -u flask-app.service -f

# 查看最近50行日志
sudo journalctl -u flask-app.service -n 50
```

## 🎯 方法二：使用 screen（可重新连接查看输出）

### 启动步骤：

```bash
# 1. SSH 连接到服务器
ssh -i "C:\Users\Abraham\Downloads\KeyPair-6418.pem" -p 2950 root@60.10.230.156

# 2. 进入项目目录
cd /var/www/html

# 3. 运行启动脚本
chmod +x start_flask_screen.sh
./start_flask_screen.sh

# 或者手动运行：
screen -dmS flask_app /usr/local/python3.11/bin/python3 app.py
```

### 管理命令：

```bash
# 查看所有 screen 会话
screen -list

# 连接到会话（查看输出）
screen -r flask_app

# 断开会话（服务继续运行）
# 在 screen 中按：Ctrl+A 然后按 D

# 停止服务
screen -S flask_app -X quit
```

## ⚡ 方法三：使用 nohup（简单快速）

### 启动步骤：

```bash
# 1. SSH 连接到服务器
ssh -i "C:\Users\Abraham\Downloads\KeyPair-6418.pem" -p 2950 root@60.10.230.156

# 2. 进入项目目录
cd /var/www/html

# 3. 运行启动脚本
chmod +x start_flask_nohup.sh
./start_flask_nohup.sh

# 或者手动运行：
nohup /usr/local/python3.11/bin/python3 app.py > flask_app.log 2>&1 &
```

### 管理命令：

```bash
# 查看日志
tail -f /var/www/html/flask_app.log

# 查看进程
ps aux | grep "python3.*app.py"

# 停止服务
pkill -f "python3.*app.py"
```

## 📋 上传文件后重启服务

### 如果使用 systemd 服务：

```bash
# 上传文件后，重启服务使更改生效
sudo systemctl restart flask-app.service

# 检查是否成功
sudo systemctl status flask-app.service
```

### 如果使用 screen：

```bash
# 1. 停止旧服务
screen -S flask_app -X quit

# 2. 重新启动
./start_flask_screen.sh
```

### 如果使用 nohup：

```bash
# 1. 停止旧服务
pkill -f "python3.*app.py"

# 2. 重新启动
./start_flask_nohup.sh
```

## 🔍 检查服务是否运行

```bash
# 方法1: 检查进程
ps aux | grep "python3.*app.py"

# 方法2: 检查端口（Flask 默认端口 5000）
netstat -tlnp | grep 5000
# 或
ss -tlnp | grep 5000

# 方法3: 测试 HTTP 访问
curl http://localhost:5000

# 方法4: 如果使用 systemd
sudo systemctl is-active flask-app.service
```

## ⚠️ 常见问题

### 1. 服务无法启动？

```bash
# 查看详细错误日志
sudo journalctl -u flask-app.service -n 50

# 检查 Python 路径是否正确
which python3
/usr/local/python3.11/bin/python3 --version

# 手动测试运行（查看错误信息）
cd /var/www/html
/usr/local/python3.11/bin/python3 app.py
```

### 2. 端口 5000 被占用？

```bash
# 查找占用端口的进程
lsof -i :5000
# 或
netstat -tlnp | grep 5000

# 停止占用进程
kill <PID>
```

### 3. 权限问题？

```bash
# 检查文件权限
ls -la /var/www/html/

# 修改权限（如果需要）
sudo chmod -R 755 /var/www/html/
```

## 🎯 推荐方案

- **生产环境**：使用 **systemd 服务**（自动重启、开机自启、统一日志管理）
- **开发/调试**：使用 **screen**（方便查看实时输出）
- **临时测试**：使用 **nohup**（快速启动）
