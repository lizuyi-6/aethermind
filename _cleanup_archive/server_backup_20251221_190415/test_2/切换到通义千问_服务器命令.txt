==========================================
  切换到通义千问 - 服务器执行命令
==========================================

在服务器上直接执行以下命令（已包含你的API密钥）：

cd /var/www/html && \
cat > .env << 'EOF'
MODEL_PROVIDER=tongyi
DASHSCOPE_API_KEY=sk-a05fb9f14d734998a8d0a7af05503058
MODEL_NAME=qwen3-max
TEMPERATURE=0.7
MAX_TOKENS=32000
EOF
&& \
pkill -f "python3 app.py" && \
python3 app.py

==========================================
