# 服务器配置说明

## 架构说明

小程序 → Flask后端服务器(app.py) → 大模型API (http://60.10.230.156:1025/v1)

小程序**不应该**直接连接大模型API，而是通过Flask后端服务器。

## 配置步骤

### 第一步：配置Flask后端服务器

确保您的Flask服务器（`app.py`）已正确配置，可以连接到您的大模型API。

#### 1.1 配置环境变量

创建 `.env` 文件或设置环境变量：

```bash
MODEL_PROVIDER=custom
CUSTOM_API_KEY=your-api-key-if-needed
API_BASE_URL=http://60.10.230.156:1025/v1
MODEL_NAME=qwen3-32b
```

#### 1.2 启动Flask服务器

```bash
python app.py
```

服务器将在 `http://localhost:5000` 启动。

### 第二步：配置小程序

#### 2.1 本地开发环境

编辑 `miniprogram/app.js`：

```javascript
globalData: {
  baseUrl: 'http://localhost:5000', // Flask服务器地址
  // ...
}
```

**注意：** 在微信开发者工具中，需要勾选"不校验合法域名"才能访问本地服务器。

#### 2.2 生产环境

如果Flask服务器部署在远程服务器上：

1. **确保服务器支持HTTPS**（小程序要求）
2. 编辑 `miniprogram/app.js`：

```javascript
globalData: {
  baseUrl: 'https://your-server.com', // 您的HTTPS服务器地址
  // ...
}
```

3. 在微信公众平台配置服务器域名

### 第三步：测试连接

#### 3.1 测试Flask服务器

```bash
curl http://localhost:5000/api/config
```

应该返回配置信息。

#### 3.2 测试小程序

1. 打开微信开发者工具
2. 导入小程序项目
3. 勾选"不校验合法域名"
4. 测试聊天功能

## 重要提示

### ⚠️ 小程序不能直接连接大模型API

小程序**必须**通过Flask后端服务器，原因：

1. **安全**：API密钥不应该暴露在前端
2. **功能**：Flask服务器提供了完整的API（文件上传、历史管理等）
3. **规范**：符合微信小程序的安全要求

### ⚠️ HTTPS要求

生产环境必须使用HTTPS：

- 申请SSL证书
- 配置服务器支持HTTPS
- 在微信公众平台配置域名

### ⚠️ 本地开发

本地开发时：
- 可以使用 `http://localhost:5000`
- 需要在微信开发者工具中勾选"不校验合法域名"
- 手机预览时，确保手机和电脑在同一网络

## 常见问题

### Q: 小程序提示"不在合法域名列表中"？

**A:** 
- 开发环境：勾选"不校验合法域名"
- 生产环境：在微信公众平台配置服务器域名

### Q: 如何让手机也能访问本地服务器？

**A:** 
1. 确保手机和电脑在同一WiFi网络
2. 将 `localhost` 改为电脑的局域网IP（如：`http://192.168.1.100:5000`）
3. 确保防火墙允许5000端口

### Q: 生产环境如何部署？

**A:**
1. 将Flask服务器部署到支持HTTPS的服务器
2. 配置域名和SSL证书
3. 在微信公众平台配置服务器域名
4. 更新小程序的 `baseUrl`

## 配置示例

### 示例1：本地开发

**Flask服务器：**
```bash
# .env
MODEL_PROVIDER=custom
API_BASE_URL=http://60.10.230.156:1025/v1
MODEL_NAME=qwen3-32b

# 启动
python app.py
```

**小程序配置：**
```javascript
// miniprogram/app.js
baseUrl: 'http://localhost:5000'
```

### 示例2：生产环境

**Flask服务器：**
- 部署到：`https://api.yourdomain.com`
- 配置HTTPS和SSL证书

**小程序配置：**
```javascript
// miniprogram/app.js
baseUrl: 'https://api.yourdomain.com'
```

**微信公众平台：**
- request合法域名：`https://api.yourdomain.com`
- uploadFile合法域名：`https://api.yourdomain.com`

## 验证配置

```bash

curl http://localhost:5000/api/config

# 2. 测试聊天API

  -H "Content-Type: application/json" \
  -d '{"message": "你好", "session_id": "test"}'

# 3. 测试大模型API（直接测试）
curl -H "Accept: application/json" \
  -H "Content-type: application/json" \
  -X POST \
  -d '{"model": "qwen3-32b", "messages": [{"role": "user", "content": "你是谁？"}]}' \
  http://60.10.230.156:1025/v1/chat/completions
```

如果所有测试都通过，说明配置正确！
