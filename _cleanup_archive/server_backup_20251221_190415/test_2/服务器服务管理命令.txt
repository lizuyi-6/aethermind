# Flask应用服务管理命令

## 方法一：systemd服务（推荐）

### 首次部署
systemctl daemon-reload
systemctl enable flask-app.service
systemctl start flask-app.service
systemctl status flask-app.service

### 日常管理
systemctl start flask-app.service      # 启动
systemctl stop flask-app.service       # 停止
systemctl restart flask-app.service    # 重启
systemctl status flask-app.service     # 查看状态
journalctl -u flask-app.service -f     # 查看日志（实时）

---

## 方法二：nohup（简单快速）

### 启动
cd /var/www/html
chmod +x start_flask_nohup.sh
./start_flask_nohup.sh

### 管理
tail -f /var/www/html/flask_app.log    # 查看日志
pkill -f "python3.*app.py"              # 停止服务
ps aux | grep "python3.*app.py"         # 查看进程

---

## 方法三：screen（可重新连接）

### 启动
cd /var/www/html
chmod +x start_flask_screen.sh
./start_flask_screen.sh

### 管理
screen -list                            # 查看所有会话
screen -r flask_app                     # 连接会话
# 在screen中按 Ctrl+A 然后按 D 断开
screen -S flask_app -X quit             # 停止服务

---

## 检查当前运行的服务

# 查找Flask进程
ps aux | grep "python3.*app.py"

# 停止所有Flask进程
pkill -f "python3.*app.py"

# 检查端口占用
netstat -tlnp | grep 5000
lsof -i:5000
