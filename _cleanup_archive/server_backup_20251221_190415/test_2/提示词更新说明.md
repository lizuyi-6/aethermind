# 提示词更新说明

## 📋 更新内容

已创建增强版提示词文件，强调以下要求：

### 1. 图表要求（15种图表类型）

报告必须包含以下所有15种不同类型的图表（使用Mermaid语法），每种图表至少使用1次：

1. **柱形图（Bar Chart）** - 用于展示分类数据对比
2. **折线图（Line Chart）** - 用于展示数据趋势
3. **饼图（Pie Chart）** - 用于展示占比关系
4. **条形图（Horizontal Bar Chart）** - 用于横向对比
5. **面积图（Area Chart）** - 用于展示累积数据
6. **XY散点图（Scatter Plot）** - 用于展示两个变量之间的关系
7. **股价图（Candlestick Chart）** - 用于展示价格波动
8. **曲面图（Surface Chart）** - 用于展示三维数据关系
9. **雷达图（Radar Chart）** - 用于多维度对比
10. **树状图（Tree Diagram）** - 用于展示层级结构
11. **旭日图（Sunburst Chart）** - 用于展示层级占比
12. **直方图（Histogram）** - 用于展示数据分布
13. **箱形图（Box Plot）** - 用于展示数据统计特征
14. **瀑布图（Waterfall Chart）** - 用于展示累积变化
15. **组合图（Combination Chart）** - 用于组合多种图表类型

### 2. 图表分布要求

每个主要章节必须包含至少2-3种不同类型的图表：

- **第一章 项目概述**：树状图、饼图、雷达图
- **第二章 项目建设背景及必要性**：折线图、柱形图、饼图
- **第三章 项目需求分析与产出方案**：条形图、面积图、散点图
- **第四章 项目选址与要素保障**：组合图、直方图、树状图
- **第五章 项目建设方案**：树状图、流程图、组合图
- **第六章 项目运营方案**：树状图、饼图、折线图
- **第七章 项目投融资与财务方案**：柱形图、折线图、瀑布图、饼图、面积图、箱形图、散点图、组合图（此章节应包含最多图表）
- **第八章 项目影响效果分析**：雷达图、柱形图、饼图
- **第九章 项目风险管控方案**：雷达图、箱形图、条形图
- **第十章 研究结论及建议**：组合图、旭日图、曲面图

### 3. 10个章节要求

报告必须包含完整的10个主要章节：

1. **第一章 项目概述**
2. **第二章 项目建设背景及必要性**
3. **第三章 项目需求分析与产出方案**
4. **第四章 项目选址与要素保障**
5. **第五章 项目建设方案**
6. **第六章 项目运营方案**
7. **第七章 项目投融资与财务方案**
8. **第八章 项目影响效果分析**
9. **第九章 项目风险管控方案**
10. **第十章 研究结论及建议**

## 🔧 使用方法

### 方法一：直接使用增强版提示词（推荐）

1. 备份原提示词文件：
   ```bash
   cp system_prompt.txt system_prompt_backup.txt
   ```

2. 使用增强版提示词：
   ```bash
   cp system_prompt_enhanced.txt system_prompt.txt
   ```

### 方法二：修改代码使用增强版提示词

修改 `config.py` 中的 `_load_system_prompt()` 方法，优先加载增强版提示词：

```python
def _load_system_prompt(self) -> str:
    # 优先从增强版文件读取
    prompt_file = 'system_prompt_enhanced.txt'
    if os.path.exists(prompt_file):
        try:
            with open(prompt_file, 'r', encoding='utf-8') as f:
                file_prompt = f.read().strip()
                if file_prompt:
                    return file_prompt
        except Exception as e:
            print(f"警告: 读取增强版提示词文件失败: {e}")
    
    # 其次从原文件读取
    prompt_file = 'system_prompt.txt'
    # ... 原有代码 ...
```
