# 超智引擎 - 项目简介

**超智引擎**是一个基于大语言模型的智能对话系统，专注于为企业用户提供可行性研究报告撰写、政策项目申报、行业数据分析等专业服务。系统采用Flask框架构建，支持Web界面、命令行界面和微信小程序三种使用方式，具备流式输出、文件上传分析、对话历史管理等完整功能。

## 核心功能

系统以"超智引擎"为智能体角色，具备以下核心能力：

1. **可行性研究报告自动生成**：支持自动加载模板、严格遵循报告格式、智能填充项目信息，可生成约50万字的详细报告，包含完整的财务分析、市场调研、技术路线等专业内容。系统具备自动续写功能，确保报告完整性和专业性。

2. **政策项目申报协助**：帮助用户准备申报材料，根据政策细则优化材料内容，提供专业的申报建议和问题解决方案。

3. **行业数据分析**：对用户提供的行业数据进行精准分析，以清晰易懂的方式呈现分析结果，为决策提供参考。

4. **政策细则解读**：用通俗易懂的语言解读政策重点内容、适用范围、实施要求等关键信息。

5. **文件智能分析**：支持多种文件格式（PDF、Word、Excel、CSV、代码文件等）的上传和分析，能够理解文件内容并回答相关问题。

## 技术特色

- **多模型支持**：兼容OpenAI API、通义千问API以及自定义OpenAI兼容API，灵活切换不同大模型服务
- **流式输出**：支持实时流式响应，提升用户体验
- **自动续写机制**：当报告生成被截断时，系统自动检测并续写，最多支持20次续写，确保内容完整
- **验证码管理系统**：内置验证码生成、验证和使用次数管理功能，支持管理员后台管理
- **多端适配**：提供Web界面、命令行工具和微信小程序，满足不同使用场景
- **对话历史管理**：支持多会话管理、历史记录保存和会话标题自动生成

## 应用场景

- 企业项目可行性研究
- 政策项目申报材料准备
- 行业数据分析和报告撰写
- 政策文件解读和咨询
- 专业文档智能分析

## 部署方式

系统支持Windows和Linux环境部署，提供一键启动脚本和详细的部署文档。支持本地部署和服务器部署，可配置为systemd服务实现后台运行。微信小程序版本需要HTTPS服务器支持。
