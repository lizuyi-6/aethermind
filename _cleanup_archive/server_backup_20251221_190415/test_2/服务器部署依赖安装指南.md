# 服务器部署 - 依赖安装指南

## 🔴 问题：找不到 requirements.txt

错误信息：
```
ERROR: Could not open requirements file: [Errno 2] No such file or directory: 'requirements.txt'
```

**原因**：`requirements.txt` 文件还没有上传到服务器。

---

## ✅ 解决方案

### 方法一：上传 requirements.txt 文件（推荐）

#### 使用 SCP 上传

在**本地电脑**上执行：

```powershell
# Windows PowerShell
scp -i ~/.ssh/KeyPair-6418.pem -P 2950 requirements.txt root@60.10.230.156:/var/www/html/
```

或者使用完整路径：
```powershell
scp -i C:\Users\Abraham\.ssh\KeyPair-6418.pem -P 2950 requirements.txt root@60.10.230.156:/var/www/html/
```

#### 使用 FTP 工具上传

1. 使用 FileZilla 或其他 FTP 工具
2. 连接到服务器
3. 上传 `requirements.txt` 到 `/var/www/html/` 目录

### 方法二：在服务器上直接创建 requirements.txt

在服务器上执行：

```bash
cd /var/www/html

# 创建 requirements.txt 文件
cat > requirements.txt << 'EOF'
openai>=1.12.0
python-dotenv>=1.0.0
PyPDF2>=3.0.0
python-docx>=1.0.0
pandas>=2.0.0
openpyxl>=3.1.0
flask>=3.0.0
flask-cors>=4.0.0
EOF
```

或者使用 nano 编辑器：

```bash
cd /var/www/html
nano requirements.txt
```

然后粘贴以下内容：
```
openai>=1.12.0
python-dotenv>=1.0.0
PyPDF2>=3.0.0
python-docx>=1.0.0
pandas>=2.0.0
openpyxl>=3.1.0
flask>=3.0.0
flask-cors>=4.0.0
```

保存：按 `Ctrl+X`，然后 `Y`，然后 `Enter`

## 📋 完整的部署步骤

### 步骤 1：确认当前目录

```bash
# 在服务器上
cd /var/www/html
pwd  # 应该显示 /var/www/html
ls -la  # 查看当前目录文件
```

### 步骤 2：上传或创建 requirements.txt

选择方法一或方法二。

### 步骤 3：验证文件存在

```bash
# 检查文件是否存在
ls -la requirements.txt

# 查看文件内容
cat requirements.txt
```

### 步骤 4：安装依赖

```bash
# 使用 python3 -m pip 安装
python3 -m pip install --upgrade pip
python3 -m pip install -r requirements.txt
```

### 步骤 5：验证安装

```bash
# 检查关键包是否安装成功
python3 -c "import flask; print('Flask:', flask.__version__)"
python3 -c "import openai; print('OpenAI: OK')"
python3 -c "import pandas; print('Pandas:', pandas.__version__)"
```

## ⚠️ 注意事项

1. **确保在正确的目录**：`/var/www/html/`
2. **检查文件权限**：`chmod 644 requirements.txt`
3. **如果使用虚拟环境**：先激活虚拟环境再安装

## 🔧 如果安装失败

### 检查 Python 版本

```bash
python3 --version
# 应该显示 Python 3.7 或更高版本
```

### 检查 pip 版本

```bash
python3 -m pip --version
```

### 逐个安装（如果批量安装失败）

```bash
python3 -m pip install openai
python3 -m pip install python-dotenv
python3 -m pip install PyPDF2
python3 -m pip install python-docx
python3 -m pip install pandas
python3 -m pip install openpyxl
python3 -m pip install flask
python3 -m pip install flask-cors
```
