# 快速配置指南

根据您提供的大模型API信息，快速配置小程序。

## 您的服务器信息

- **大模型API地址：** `http://60.10.230.156:1025/v1/chat/completions`
- **模型名称：** `qwen3-32b`

## 配置步骤（3步完成）

### 步骤1：配置Flask后端服务器

编辑项目根目录的 `.env` 文件（如果没有则创建）：

```bash
MODEL_PROVIDER=custom
API_BASE_URL=http://60.10.230.156:1025/v1
MODEL_NAME=qwen3-32b
CUSTOM_API_KEY=your-api-key-if-needed  # 如果需要API密钥
```

或者直接设置环境变量：

**Windows:**
```cmd
set MODEL_PROVIDER=custom
set API_BASE_URL=http://60.10.230.156:1025/v1
set MODEL_NAME=qwen3-32b
```

**Linux/Mac:**
```bash
export MODEL_PROVIDER=custom
export API_BASE_URL=http://60.10.230.156:1025/v1
export MODEL_NAME=qwen3-32b
```

### 步骤2：启动Flask服务器

```bash
python app.py
```

服务器将在 `http://localhost:5000` 启动。

**验证服务器是否正常：**
```bash
curl http://localhost:5000/api/config
```

应该返回配置信息。

### 步骤3：配置小程序

编辑 `miniprogram/app.js`，找到 `globalData`：

```javascript
globalData: {
  baseUrl: 'http://localhost:5000', // 本地开发
  // 如果Flask服务器在其他地址，改为：
  // baseUrl: 'http://your-server-ip:5000',
  // ...
}
```

## 常见问题

### Q: 小程序提示"不在合法域名列表中"？

**A:** 本地开发时，在微信开发者工具中勾选"不校验合法域名"即可。

### Q: Flask服务器启动失败？

**A:** 
1. 检查Python环境
2. 检查依赖是否安装：`pip install -r requirements.txt`
3. 检查端口5000是否被占用

### Q: API调用失败？

**A:**
1. 检查Flask服务器是否运行
2. 检查 `.env` 配置是否正确
3. 检查大模型API是否可访问：
   ```bash
   curl -H "Accept: application/json" \
     -H "Content-type: application/json" \
     -X POST \
     -d '{"model": "qwen3-32b", "messages": [{"role": "user", "content": "你好"}]}' \
     http://60.10.230.156:1025/v1/chat/completions
   ```

4. **更新小程序的baseUrl为HTTPS地址**
