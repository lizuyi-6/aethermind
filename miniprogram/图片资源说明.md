# 图片资源说明

小程序需要以下图片资源，请放置在 `miniprogram/images/` 目录下：

## 需要的图片

### TabBar 图标

需要在 `miniprogram/images/` 目录下创建以下图标文件：

1. **home.png** - 首页未选中图标
   - 尺寸：81px × 81px
   - 颜色：灰色 (#999999)
   - 建议使用简单的房子图标

2. **home-active.png** - 首页选中图标
   - 尺寸：81px × 81px
   - 颜色：绿色 (#4CAF50)
   - 与 home.png 相同设计，但使用选中颜色

3. **chat.png** - 对话未选中图标
   - 尺寸：81px × 81px
   - 颜色：灰色 (#999999)
   - 建议使用对话气泡图标

4. **chat-active.png** - 对话选中图标
   - 尺寸：81px × 81px
   - 颜色：绿色 (#4CAF50)
   - 与 chat.png 相同设计，但使用选中颜色

## 图标制作建议

### 方式一：使用在线图标库

推荐网站：
- [IconFont](https://www.iconfont.cn/) - 阿里巴巴图标库
- [Flaticon](https://www.flaticon.com/) - 免费图标库
- [Icons8](https://icons8.com/) - 图标库

搜索关键词：
- 首页：home, house, 首页
- 对话：chat, message, 对话, 消息

### 方式二：使用设计工具

使用 Figma、Sketch 或 Photoshop 制作：
- 尺寸：81px × 81px
- 格式：PNG（透明背景）
- 设计风格：简洁、现代

### 方式三：临时使用文字图标

如果暂时没有图片，可以修改 `app.json`，使用文字代替：

```json
{
  "tabBar": {
    "list": [
      {
        "pagePath": "pages/index/index",
        "text": "首页"
        // 移除 iconPath 和 selectedIconPath
      },
      {
        "pagePath": "pages/chat/chat",
        "text": "对话"
        // 移除 iconPath 和 selectedIconPath
      }
    ]
  }
}
```

## 快速开始

如果暂时没有图标，可以：

1. 从 [IconFont](https://www.iconfont.cn/) 下载免费图标
2. 或者先移除 TabBar 图标配置，只使用文字
3. 或者使用简单的 emoji 作为占位符

## 目录结构

创建后的目录结构应该是：

```
miniprogram/
├── images/
│   ├── home.png
│   ├── home-active.png
│   ├── chat.png
│   └── chat-active.png
├── app.json
└── ...
```
