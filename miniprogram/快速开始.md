# 快速开始指南

5分钟快速上手AetherMind微信小程序。

## 第一步：准备环境

1. **安装微信开发者工具**
   - 下载：https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html
   - 安装并登录

2. **注册小程序账号**
   - 访问：https://mp.weixin.qq.com/
   - 注册并获取 AppID

3. **确保后端服务器运行**
   - 确保 Flask 服务器（app.py）正在运行
   - 确保服务器支持 HTTPS（生产环境必需）

## 第二步：配置小程序

### 2.1 修改服务器地址

打开 `miniprogram/app.js`，修改：

```javascript
globalData: {
  baseUrl: 'https://your-server.com', // 改为您的服务器地址
  // 本地开发可以使用：'http://localhost:5000'（需要勾选不校验域名）
}
```

### 2.2 配置 AppID

打开 `miniprogram/project.config.json`，修改：

```json
{
  "appid": "your-appid-here" // 改为您的 AppID
}
```

## 第三步：打开项目

1. 打开微信开发者工具
2. 选择"导入项目"
3. 选择 `miniprogram` 目录
4. 输入 AppID（可以使用测试号）
5. 点击"导入"

## 第四步：开发环境设置

在微信开发者工具中：

1. 点击右上角"详情"
2. 勾选"不校验合法域名、web-view（业务域名）、TLS 版本以及 HTTPS 证书"
   - 这样可以在开发时访问本地服务器

## 第五步：测试

1. 点击"编译"按钮
2. 测试以下功能：
   - 发送消息
   - 文件上传
   - 查看对话历史

## 第六步：配置服务器域名（生产环境）

1. 登录微信公众平台
2. 进入"开发" -> "开发管理" -> "开发设置"
3. 配置服务器域名：
   - request合法域名：`https://your-server.com`
   - uploadFile合法域名：`https://your-server.com`

## 常见问题

### Q: 提示"不在合法域名列表中"？

**A:** 开发环境可以勾选"不校验合法域名"。生产环境需要在微信公众平台配置服务器域名。

### Q: 无法连接服务器？

**A:** 
1. 检查服务器是否运行
2. 检查 baseUrl 是否正确
3. 开发环境勾选"不校验合法域名"

### Q: 文件上传失败？

**A:**
1. 检查文件大小（不能超过50MB）
2. 检查文件类型是否支持
3. 检查服务器 uploadFile 域名配置
